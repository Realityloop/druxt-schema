# DruxtSchemaMixin

Lazy loads a schema into the Vuex State object.

Kind: global mixin
Todo

  • Add schemaType property to Mixin.

Example

<script>
// Import mixin.
import { DruxtSchemaMixin } from 'druxt-schema'

export default {
  name: 'CustomComponent',

  // Register mixin.
  mixins: [DruxtSchemaMixin],
}
</script>
